<p>Instructions for adding this repo:</p>
<ul>
<li>Go to the Kodi file manager</li>
<li>Click on "Add source"</li>
<li>The path for the source is <code class="code-highlight">https://kpoodle.github.io/repo</code></li>
<li>Name it "Poodle" without the quotes</li>
<li>Head to " My Addons"</li>
<li>Select install an addon from zipfile. </li>
<li>When it asks for the location, select "Poodle", and install <code class="code-highlight">repository.PoodleWiz-1.0.0.zip</code>.</li>
<li>Go back to My Addons and select "Install from repository"</li>
<li>Select the "Poodle" menu item</li>
<li>Open "Program add-ons" and install Poodle Wizard - for v18 or v19</li>
<li>(Optional) It's After tha you can install Poodle build for for v18 or v19</li>
</ul>
