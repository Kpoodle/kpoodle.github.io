Installing the Poodle Wizard Addon Current Repository Version: 1.0.0
Go to the Kodi file manager
Click on "Add source"
The path for the source is: https://kpoodle.github.io/repo
Name it "Poodle" without the quotes
Head to " My Addons"
Select install an addon from zipfile.
When it asks for the location, select "Poodle", and install: repository.PoodleWiz-1.0.0.zip
Go back to My Addons and select "Install from repository"
Select the "Poodle" menu item
Open "Program add-ons" and install Poodle Wizard - for v18 or v19
(Optional) It's After tha you can install Poodle build for for v18 or v19
